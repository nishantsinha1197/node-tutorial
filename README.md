Node tutorial is covered in different folder in a systematic manner like 1,2,3,4 so on.

1. Folder 1 - Modules - It is about the modules used in node like os,path,fs and http. In fs module synchronous and asynchronous methods of reading and writing file is covered . In http module , the way  node.js transfers data over HTTP (Hyper Text Transfer Protocol) is covered.

2. Folder 2 - Content - It is basically the text files which were used while reading and writing data using modules.

3. Folder 3 - lodash - In this  how to flatten items using lodash is shown

4. Folder 4 - EventLoop - In this all the methods the challenges of handling event is covered by using async , promise , util, promises is shown
